Regularized scVI with ambient RNA correction and overdispersion regularisation, based on cell2location modelling principles

## Getting started

Please refer to the [documentation][],
in particular, the [API documentation][].

## Installation

You need to have Python 3.11 or newer installed on your system.
If you don't have Python installed, we recommend installing [uv][].

There are several alternative options to install regularizedvi:

<!--
1) Install the latest release of `regularizedvi` from [PyPI][]:

```bash

```
-->

1. Install the latest development version:

```bash
pip install git+https://github.com/vitkl/regularizedvi.git@main
```
